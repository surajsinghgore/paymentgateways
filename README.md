## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`STRIPE_SCERET_API_KEY` 

`STRIPE_PUBLISHABLE_KEY`

`RAZORPAY_KEY`

`RAZORPAY_ID`

`PAYTM_MERCHANT_KEY`

`PAYTM_MERCHANT_ID`

`MONGODB_URL`

`PAYTM_HOST`
